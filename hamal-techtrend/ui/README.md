# Hamal UI

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the backend server (in root directory):
   ```bash
   python hamal.py serve
   ```

3. Start the UI development server:
   ```bash
   npm run dev
   ```

4. Open browser to http://localhost:5173

## Features
- Connects to Hamal Agent Backend
- Real-time progress tracking
- Markdown and HTML Report viewing
