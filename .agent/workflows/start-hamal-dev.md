---
description: Start the Hamal AI local development environment detailed
---

1.  **Start Backend**
    *   Open terminal named "Backend"
    *   Navigate to `hamal-ai/backend`
    *   Run: `source venv/bin/activate && uvicorn main:app --reload`

2.  **Start Frontend**
    *   Open terminal named "Frontend"
    *   Navigate to `hamal-ai/frontend`
    *   Run: `npm run dev`

3.  **Verify**
    *   Check that Backend is running on port 8000
    *   Check that Frontend is open/running on localhost (usually 5173)
