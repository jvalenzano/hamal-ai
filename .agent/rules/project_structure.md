# Project Structure & Organization

## Directory Standards
*   **Framework Docs**: Hamal methodology lives in `hamal-framework/` (phases 01-05).
*   **Ideas & Concepts**: Raw ideas and V2 planning in `Jasons_Ideas/`.
*   **Prototypes**: Functional code in `hamal-ai/` (backend + frontend).
*   **Ventures**: Validated ideas graduate to `ventures/[name]/`.

## File Naming
*   **Python**: Snake_case (`main.py`, `data_processor.py`).
*   **React**: PascalCase for components (`AgentStatus.tsx`), camelCase for logic (`apiClient.ts`).
*   **Docs**: Kebab-case/descriptive names (`echo-research.md`).

## Key Files
*   `NORTH_STAR.md` - Project vision and strategy
*   `_FOCUS.md` - Current sprint objectives
*   `KEEP_IT_SIMPLE.md` - POC development manifesto
